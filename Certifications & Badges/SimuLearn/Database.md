Databases in Practice
----
Completed certification covering core database fundamentals, including data models, ACID principles, high availability, and backup concepts.

----
Key implementations: 

- Relational databases (SQL): structured schemas, keys, normalization, and transactional workloads (Amazon RDS, Aurora).
- Non-relational databases (NoSQL): flexible schemas, key-value and document models, and scalable workloads (Amazon DynamoDB).
- AWS managed database concepts: automated backups, patching, Multi-AZ deployments, read replicas, encryption, IAM, and VPC integration.
